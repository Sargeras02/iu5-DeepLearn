# Требования к оформлению отчётов

Оформления отчёта по ГОСТ 2.105-95. Общие обязательные структурные элементы:
- СОДЕРЖАНИЕ
- ЗАКЛЮЧЕНИЕ _(не 'ВЫВОДЫ'!)_

Должны быть отражены цели и задачи проделываемой работы.

Краткая выжимка по ГОСТ в 'Приложение к МР 2025' - для удобства пользования.
Сверяйте требования по актуальному стандарту.

Все таблицы и рисунки должны быть пронумерованы. На все таблицы и рисунки должна быть сслыка в тексте отчёта.

Сводная таблица по всем экспериментам приводится в конце, перед ЗАКЛЮЧЕНИЕМ.

Общие требования по сдаче отчётов на [главной странице курса](../../../).
