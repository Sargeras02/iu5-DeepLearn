Веб-лаборатория ONNX Vision

Учебно-демонстрационный стенд для визуализации работы нейронных сетей в режиме "на лету" с использованием пользовательских ONNX-моделей

Порядок работы:
1. На первом шаге загрузить ONNX-модель (формат Dynamo не поддерживается), задать размерность входных каналов и выбрать layout данных
2. На втором шаге определить метки классов в соответствии с выбранным вариантом
3. На третьем шаге загрузить входное изображение для выполнения инференса
Ограничения реализации: возможны переполнения WASM-памяти; в таком случае требуется перезагрузка браузера (ограничение Web ONNX Runtime)
