# Лабораторный стенд проверки ONNX моделей

Инструкция:
1. Скачать `/single/index.html`
2. Открыть в браузере

Проверено на:
1. Firefox (152.0 64-разрядный)
2. Microsoft Edge (149.0.4022.80 64-разрядный)
3. Google Chrome (149.0.7827.156 64-разрядный)

Требования ко входу модели:
- Схема входа: NCHW
- Входной тензор: (1, 3, 32, 32)


# Структура папки

- `/src` - исходный код, разбитый на файлы; предпочтительно для модификации
- `/single` - плоская проекция src, сведённая в один файл; предпочтительно для тестирования ONNX
